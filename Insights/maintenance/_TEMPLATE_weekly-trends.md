## 주간 트렌드 합성 작성 가이드 (에이전트용)

> 이 섹션은 AI 에이전트가 주간 트렌드 합성을 자동 생성할 때 참조하는 지침입니다.

### 작성 프로세스

1. 이번 주 Daily Digest 모든 기사를 읽습니다
2. 각 기사의 주제를 기존 Insights 카테고리에 매핑합니다
3. **매핑 실패 기사가 3건 이상**이면 아래 형식으로 플래깅합니다:

```
### 신규 카테고리 후보 감지
- 매핑 실패 기사 수: X건
- 공통 주제 키워드: [키워드1], [키워드2], ...
- 제안 카테고리 slug: [제안-slug]
- 제안 카테고리 설명: [설명]
```

4. 카테고리별로 주요 동향을 3-5개 bullet point로 정리합니다
5. 제품별 업데이트를 테이블에 기록합니다 (영향도: 높음/중간/낮음)
6. **전략적 시사점**을 작성합니다:
   - 영향도 '높음' 기사 중 제품 개발/전략에 직접 영향을 미치는 항목을 분류
   - "즉시 대응": 이번 스프린트에서 반영해야 할 항목 (경쟁사 신기능 출시, 시장 변화 등)
   - "로드맵 반영 검토": 중기적으로 로드맵에 반영을 검토할 항목
   - "모니터링 계속": 아직 대응 불필요하지만 추이를 지켜볼 항목 + 대응 트리거 조건 명시
   - 각 Action Item에 담당 역할(AGENTS.md 기준)과 관련 인사이트 문서를 연결
7. 다음 주 예정된 이벤트를 리서치하여 "다음 주 주목 포인트"에 기록합니다

### 인용 규칙
- 모든 팩트성 서술에 각주 `[^N]` 인라인 출처 필수
- References 섹션에 각주 번호별 클릭 가능한 출처 링크 기재
