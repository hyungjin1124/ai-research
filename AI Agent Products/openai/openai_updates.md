### 2026-02-10 | ChatGPT 광고 도입 시작

**소스**: [Testing ads in ChatGPT](https://openai.com/index/testing-ads-in-chatgpt)
**요약**: OpenAI가 ChatGPT 내 광고 테스트를 시작했다. 무료 접근 지원을 위한 수익화 전략으로, 명확한 라벨링·답변 독립성·개인정보 보호·사용자 제어 기능을 갖췄다. AI 챗봇 비즈니스 모델의 중대한 전환점이다.
**카테고리**: 신기능
**영향도**: 높음

### 2026-02-11 | arXiv: 병렬 도구 호출 스케일링 W&D — GPT-5-Medium BrowseComp 62.2%
**소스**: [W&D: Scaling Parallel Tool Calling for Efficient Deep Research Agents](https://arxiv.org/abs/2602.07359)
**요약**: 병렬 도구 호출 스케일링 프레임워크 W&D가 GPT-5-Medium으로 BrowseComp에서 62.2% 정확도 달성. 에이전트 도구 호출 폭 확장이 Deep Research 에이전트 효율성에 기여.
**카테고리**: 기술업데이트
**영향도**: 보통

### 2026-02-12 | Mission Alignment Team 해산 — AI 안전 조직 재편 지속
**소스**: [OpenAI disbands mission alignment team](https://techcrunch.com/2026/02/11/openai-disbands-mission-alignment-team-which-focused-on-safe-and-trustworthy-ai-development/)
**요약**: OpenAI가 Mission Alignment Team을 해산하고 리더 Josh Achiam에게 'Chief Futurist' 직함을 부여. 2024년 Superalignment Team에 이어 두 번째 안전팀 해체로, AI 안전 커뮤니티에서 우려 제기.
**카테고리**: 기타
**영향도**: 보통

### 2026-02-12 | Skills API + Shell Tool + Server-side Compaction 발표
**소스**: [OpenAI upgrades Responses API with Skills and Shell](https://venturebeat.com/orchestration/openai-upgrades-its-responses-api-to-support-agent-skills-and-a-complete)
**요약**: Responses API에 Skills(SKILL.md 매니페스트 기반 재사용 모듈), Hosted Shell(Debian 12 컨테이너 코드 실행), Server-side Compaction(컨텍스트 자동 압축)을 추가. Anthropic MCP 생태계와 직접 경쟁하는 에이전트 인프라 전략.
**카테고리**: 신기능
**영향도**: 높음
